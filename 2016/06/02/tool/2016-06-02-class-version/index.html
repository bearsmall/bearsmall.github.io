查看.class文件的版本号

发表于 2016-06-02 | 分类于 工具 |

工具

步骤

  1. 用编辑器打开.class文件
  2. 可以看到如下内容：

解释：

  1. 我们只关心第一行画线的部分。
  2. 前面八个字节CA FE BA BE是固定的。
  3. 随后四个字节00 00是次版本号。
  4. 再随后四个字节00 34是主版本号【这里是1.8】

JDK版本号对应关系如下：

  • 34（对应十进制的52）：JDK1.8
  • 33（对应十进制的51）：JDK1.7
  • 32（对应十进制的50）：JDK1.6
  • 31（对应十进制的49）：JDK1.5
  • 30（对应十进制的48）：JDK1.4
  • 2F（对应十进制的47）：JDK1.3
  • 2E（对应十进制的46）：JDK1.2
